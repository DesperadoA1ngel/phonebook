# PhoneBook Application

## Описание

PhoneBook - это приложение для управления телефонной книгой, разработанное на C++ с использованием фреймворка Qt. Программа предоставляет возможности для добавления, редактирования, удаления и поиска контактов с возможностью сортировки по различным полям.

## Требования

- **Операционная система:** Windows 10/11 или Linux
- **Компилятор:** MinGW 8.1.0 или выше (Windows), GCC (Linux)
- **Фреймворк:** Qt 5.12.5 или выше
- **Система сборки:** CMake 3.16 или выше

## Функциональность

### Основные возможности:
- 📋 Отображение контактов в табличном виде
- ➕ Добавление новых контактов
- ✏️ Редактирование существующих контактов
- ❌ Удаление контактов
- 🔍 Поиск по всем полям контакта
- 🔄 Сортировка по любому столбцу (ID, ФИО, Email, Дата рождения, Дата добавления)
- 📊 10 предустановленных тестовых контактов
- 🔙 Возможность сброса к исходным данным

### Поля контакта:
1. **ID** - порядковый номер
2. **Full Name** - ФИО контакта
3. **Email** - электронная почта
4. **Birthday** - дата рождения
5. **Added Date** - дата внесения в книгу

## Сборка и запуск

### Для Windows:

1. **Установите зависимости:**
   - Qt 5.15.2 (MinGW 32-bit)
   - CMake 3.16+
   - MinGW 8.1.0

2. **Соберите проект:**
   ```bash
   mkdir build
   cd build
   cmake -G "MinGW Makefiles" -DCMAKE_PREFIX_PATH="C:/Qt5/5.15.2/mingw81_32/lib/cmake" ..
   mingw32-make
   ```

3. **Скопируйте необходимые DLL:**
   ```bash
   copy "C:\Qt5\5.15.2\mingw81_32\bin\Qt5Core.dll" .
   copy "C:\Qt5\5.15.2\mingw81_32\bin\Qt5Widgets.dll" .
   copy "C:\Qt5\5.15.2\mingw81_32\bin\Qt5Gui.dll" .
   mkdir platforms
   copy "C:\Qt5\5.15.2\mingw81_32\plugins\platforms\qwindows.dll" platforms\
   ```

4. **Запустите приложение:**
   ```bash
   PhoneBook.exe
   ```

### Для Linux:

1. **Установите зависимости:**
   ```bash
   sudo apt-get update
   sudo apt-get install build-essential cmake qt5-default
   ```

2. **Соберите проект:**
   ```bash
   mkdir build
   cd build
   cmake ..
   make
   ```

3. **Запустите приложение:**
   ```bash
   ./PhoneBook
   ```

### Использование скриптов (Windows):

В папке проекта создайте скрипты:

**build.bat** - для сборки:
```batch
@echo off
if exist build rmdir /s /q build
mkdir build
cd build
cmake -G "MinGW Makefiles" -DCMAKE_PREFIX_PATH="C:/Qt5/5.15.2/mingw81_32/lib/cmake" ..
mingw32-make
cd ..
pause
```

**launch.bat** - для запуска:
```batch
@echo off
cd build
start PhoneBook.exe
```

## Использование программы

### Интерфейс:
1. **Верхняя часть:** Таблица с контактами
2. **Средняя часть:** Форма для ввода/редактирования данных
3. **Нижняя часть:** Панель кнопок и поле поиска

### Основные операции:

#### Добавление контакта:
1. Заполните поля в форме
2. Нажмите кнопку "Добавить"
3. Контакт появится в таблице

#### Редактирование контакта:
1. Дважды кликните по контакту в таблице
2. Данные загрузятся в форму
3. Измените нужные поля
4. Нажмите кнопку "Изменить"

#### Удаление контакта:
1. Выберите контакт в таблице
2. Нажмите кнопку "Удалить"
3. Подтвердите удаление

#### Поиск контактов:
1. Введите текст в поле поиска
2. Нажмите кнопку "Найти" или Enter
3. Таблица отфильтруется по введенному тексту

#### Сортировка:
1. Нажмите на заголовок любого столбца
2. Для обратной сортировки нажмите еще раз

#### Сброс данных:
1. Нажмите кнопку "Сброс"
2. Подтвердите действие
3. Данные вернутся к исходным 10 контактам

## Классы программы

### 1. Contact
Класс для хранения данных одного контакта:
- ID, ФИО, Email, даты рождения и добавления
- Геттеры и сеттеры для всех полей

### 2. PhoneBook
Класс-модель для управления коллекцией контактов:
- CRUD операции (Create, Read, Update, Delete)
- Поиск и сортировка
- Загрузка тестовых данных

### 3. MainWindow
Главное окно приложения:
- Пользовательский интерфейс
- Обработка событий
- Взаимодействие с моделью данных

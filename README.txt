PhoneBook Application

Описание проекта

PhoneBook - это кроссплатформенное приложение для управления контактами, разработанное с использованием Qt5 и C++17. Приложение предоставляет удобный интерфейс для хранения и управления контактами с возможностью поиска, сортировки и редактирования.

Функциональность

Основные возможности:
- Отображение контактов в табличном виде с 5 столбцами
- CRUD операции: создание, чтение, обновление, удаление контактов
- Поиск по всем полям контакта
- Сортировка по любому столбцу (ID, ФИО, Email, Дата рождения, Дата добавления)
- Тестовые данные: 10 предустановленных контактов
- Сброс данных к исходному состоянию
- Кроссплатформенность: работает на Windows и Linux

Поля контакта:
1. ID - уникальный идентификатор
2. Full Name - полное имя контакта
3. Email - электронная почта
4. Birthday - дата рождения
5. Added Date - дата добавления в книгу

Системные требования

Для Windows:
- ОС: Windows 10/11 (64-bit)
- Компилятор: MinGW 8.1.0 или выше
- Фреймворк: Qt 5.12.5 или выше (рекомендуется Qt 5.15.2 MinGW 32-bit)
- Система сборки: CMake 3.16 или выше
- Дополнительно: MinGW Toolchain (gcc, g++, make)

Для Linux:
- Дистрибутив: Ubuntu 20.04+, Fedora, Debian и др.
- Компилятор: GCC 9.0 или выше
- Фреймворк: Qt5 Development Tools
- Система сборки: CMake 3.16+

Сборка проекта

Windows (используя предоставленные скрипты)

Вариант 1: Стандартная сборка (build.bat)
```bash
# Запустите в корневой папке проекта:
build.bat
```
Скрипт выполнит:
1. Создание папки `build` (если не существует)
2. Конфигурацию CMake
3. Сборку проекта
4. Создание скрипта `deploy_qt.bat` для копирования DLL

Вариант 2: Полная очистка и сборка (clean.bat)
```bash
# Для полной пересборки:
clean.bat
```
Скрипт выполнит:
1. Удаление старой папки `build`
2. Создание новой папки сборки
3. Полную конфигурацию и сборку
4. Создание скрипта развертывания

Вариант 3: Ручная сборка
```bash
# Создайте папку сборки
mkdir build
cd build

Сконфигурируйте проект
cmake .. -G "MinGW Makefiles" -DCMAKE_PREFIX_PATH="C:/Qt5/5.15.2/mingw81_32/lib/cmake"

Соберите проект
mingw32-make
```

Linux
```bash
# Создайте папку сборки
mkdir build
cd build

Сконфигурируйте проект

Соберите проект
make
```

Развертывание и запуск

Windows

После успешной сборки:

1. Скопируйте необходимые DLL файлы:
   ```bash
   cd build
   deploy_qt.bat
   ```
   Или вручную:
   ```bash
  Основные Qt DLL
   copy "C:\Qt5\5.15.2\mingw81_32\bin\Qt5Core.dll" .
   copy "C:\Qt5\5.15.2\mingw81_32\bin\Qt5Widgets.dll" .
   copy "C:\Qt5\5.15.2\mingw81_32\bin\Qt5Gui.dll" .
   
   Runtime DLL для MinGW
   copy "C:\Qt5\5.15.2\mingw81_32\bin\libgcc_s_dw2-1.dll" .
   copy "C:\Qt5\5.15.2\mingw81_32\bin\libstdc++-6.dll" .
   copy "C:\Qt5\5.15.2\mingw81_32\bin\libwinpthread-1.dll" .
   
   Плагины Qt
   mkdir platforms
   copy "C:\Qt5\5.15.2\mingw81_32\plugins\platforms\qwindows.dll" platforms\
   ```

2. Запустите приложение:
   ```bash
   PhoneBook.exe
   ```

Linux
```bash
# После сборки просто запустите
cd build
./PhoneBook
```

Использование приложения

Интерфейс приложения:
1. Верхняя часть: Таблица со всеми контактами
2. Средняя часть: Форма для ввода/редактирования данных контакта
3. Нижняя часть: Панель управления (кнопки) и поле поиска

Основные операции:

Добавление контакта:
1. Заполните поля в форме (ФИО, Email, Дата рождения)
2. Нажмите кнопку "Добавить"
3. Новый контакт появится в таблице

Редактирование контакта:
1. Дважды кликните по контакту в таблице
2. Данные загрузятся в форму редактирования
3. Измените необходимые поля
4. Нажмите кнопку "Изменить"

Удаление контакта:
1. Выберите контакт в таблице (кликните по строке)
2. Нажмите кнопку "Удалить"
3. Подтвердите удаление в диалоговом окне

#### Поиск контактов:
1. Введите текст в поле поиска внизу окна
2. Нажмите кнопку "Поиск" или клавишу Enter
3. Таблица отобразит только контакты, соответствующие запросу

Сортировка:
1. Нажмите на заголовок любого столбца таблицы
2. Для обратной сортировки нажмите на заголовок еще раз

Сброс данных:
1. Нажмите кнопку "Сброс"
2. Подтвердите действие в диалоговом окне
3. Все изменения будут удалены, вернутся исходные 10 контактов

Настройка путей

Если у вас Qt установлен в другом месте, отредактируйте пути в скриптах:

В `build.bat` и `clean.bat`:
```batch
# Измените эту строку на ваш путь к Qt
set QT_PATH=C:\Qt5\5.15.2\mingw81_32
```

Или укажите путь при запуске CMake:
```bash
cmake .. -G "MinGW Makefiles" -DCMAKE_PREFIX_PATH="ВАШ_ПУТЬ_К_QT/lib/cmake"
```

Устранение проблем

Проблема: "Unable to find the platform plugin"
Решение: Убедитесь, что папка `platforms` создана и содержит `qwindows.dll`

Проблема: "Missing Qt5Core.dll"
Решение: Запустите `deploy_qt.bat` или скопируйте DLL файлы вручную

Проблема: Ошибка сборки
Решение: 
1. Убедитесь, что установлены все зависимости
2. Запустите `clean.bat` для полной пересборки
3. Проверьте правильность пути к Qt

Проблема: Приложение не запускается
Решение: 
1. Проверьте наличие всех необходимых DLL файлов в папке с `PhoneBook.exe`
2. Убедитесь, что архитектура Qt соответствует архитектуре компилятора (32-bit MinGW)
